STEP 4: ENABLE HTTPS ON HOSTINGER (5 minutes)
═══════════════════════════════════════════════════════════════════════

CRITICAL: APK requires HTTPS (SSL certificate)

□ Login to Hostinger hPanel

□ Go to: Security → SSL

□ Click "Install SSL" or "Enable SSL"
  (Free SSL included with Hostinger)

□ Enable "Force HTTPS" option
  (Redirects HTTP to HTTPS automatically)

□ Wait 5-10 minutes for SSL to activate

□ Clear browser cache and visit your site

Verify:
- Open your website
- URL should show: https://yourdomain.com (with padlock 🔒)
- Not: http://yourdomain.com
- Click padlock → Should say "Connection is secure"

STEP 5: ADD PWA CODE TO YOUR LOGIN PAGE (5 minutes)
═══════════════════════════════════════════════════════════════════════

□ Open Hostinger File Manager

□ Navigate to: public_html/admin/login.php

□ Click "Edit" button

□ Find the <head> section (around line 58-66)

□ After the viewport meta tag, add this line:

  <?php include '../includes/pwa_head.php'; ?>

  Example:
  <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <?php include '../includes/pwa_head.php'; ?>  ← ADD THIS LINE
      <title>...

□ Click "Save"

□ Repeat for other main pages:
  - admin/dashboard.php
  - modules/students/list.php
  - modules/fee_collection/collect.php

Verify:
- Open your website
- Right-click → View Page Source
- Search for "manifest.json"
- Should find: <link rel="manifest" href="/manifest.json">

STEP 7: CREATE APK FILE (5 MINUTES!) 🎉
═══════════════════════════════════════════════════════════════════════

NOW you can create the APK!

METHOD A: PWABuilder (Easiest - Recommended)
───────────────────────────────────────────────────────────────────────

□ Go to: https://www.pwabuilder.com/

□ Enter your website URL: https://yourdomain.com

□ Click "Start" button

□ Wait 30 seconds for analysis

□ You should see green checkmarks for:
  ✓ Manifest
  ✓ Service Worker
  ✓ HTTPS

□ Click "Package For Stores" button

□ Click "Android" or "Google Play"

□ Fill in the form:
  - Package ID: com.yourschool.feemgmt
    (Example: com.stjohns.feemgmt)

  - App name: Fee Management System
    (Or: "Your School Name - Fees")

  - Version: 1.0.0

  - Version code: 1

  - Host: yourdomain.com

  - Start URL: /admin/login.php

□ Click "Generate Package" or "Download"

□ Wait 2-5 minutes for APK generation

□ Download APK file (FREE!)

□ You'll get: app-release.apk or similar file

Verify:
✓ APK file downloaded (size: 1-5 MB)
✓ File type: .apk
✓ Ready to install!

METHOD B: If PWABuilder Shows Errors
───────────────────────────────────────────────────────────────────────

If PWABuilder says "Manifest not found" or similar:

1. Go back to Step 3 and verify files are uploaded correctly
2. Check: https://yourdomain.com/manifest.json (should work)
3. Check: https://yourdomain.com/sw.js (should work)
4. Wait 5 minutes and try again (DNS cache)

6. Try PWABuilder again

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

Problem: PWABuilder says "Manifest not found"
Solution:
□ Verify: https://yourdomain.com/manifest.json works
□ Check file is uploaded to root directory (not subdirectory)
□ Wait 5-10 minutes (DNS cache)
□ Clear browser cache

Problem: "Install app" doesn't show on phone
Solution:
□ Verify HTTPS is enabled (padlock icon)
□ Use Chrome browser (not Safari, Firefox)
□ Check manifest.json is accessible
□ Clear Chrome cache
□ Try in Incognito mode

Problem: APK won't install
Solution:
□ Enable "Install from unknown sources"
□ Check Android version (need Android 5.0+)
□ APK file not corrupted (re-download)
□ Enough storage space on phone

Problem: App opens but blank screen
Solution:
□ Check start_url in manifest.json
□ Verify website is accessible
□ Check HTTPS is working
□ Clear app data and try again
