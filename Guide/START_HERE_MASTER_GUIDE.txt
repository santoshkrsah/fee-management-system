   - Risk: Users stay logged in forever
   - Impact: Security vulnerability

2. ❌ **Weak Password Policy:** Only 6 characters minimum
   - Risk: Easy to guess passwords
   - Impact: Brute force attacks possible

3. ❌ **No Login Rate Limiting:** Unlimited login attempts
   - Risk: Brute force attacks
   - Impact: Account compromise

4. ❌ **No Audit Trail:** Actions not logged
   - Risk: No accountability
   - Impact: Cannot track who did what

5. ❌ **Password Reset:** NOT IMPLEMENTED
   - Risk: Locked out users need admin help
   - Impact: Poor user experience

6. ❌ **Change Password:** NOT IMPLEMENTED
   - Risk: Users can't change their own passwords
   - Impact: Requires admin intervention

7. ❌ **HTTPS Not Enforced:** Cookies can be intercepted
   - Risk: Session hijacking
   - Impact: Security vulnerability
