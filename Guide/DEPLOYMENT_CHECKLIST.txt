□ STEP 1: HOSTINGER ACCOUNT
  □ Have Hostinger login credentials ready
  □ Know your domain name (e.g., myschool.com)

□ STEP 2: CREATE DATABASE
  □ Login to Hostinger hPanel
  □ Go to: Databases → MySQL Databases
  □ Click "Create New Database"
  □ Write down these details:

    Database Name: _________________________________
    Database User: _________________________________
    Database Pass: _________________________________

□ STEP 3: IMPORT DATABASE
  □ Open phpMyAdmin from hPanel
  □ Select your database
  □ Import file: database_complete_with_students.sql
  □ Verify 50 students are loaded

□ STEP 4: UPLOAD FILES
  □ Go to: Files → File Manager
  □ Open public_html folder
  □ Delete default files
  □ Upload ALL files from fee_management_system folder
    Files to upload:
    □ index.php
    □ admin folder
    □ assets folder
    □ config folder
    □ includes folder
    □ modules folder
    □ database_complete_with_students.sql (for backup)

□ STEP 5: UPDATE CONFIG FILE
  □ Open: public_html/config/database.php
  □ Update DB_HOST: localhost
  □ Update DB_NAME: (your database name from Step 2)
  □ Update DB_USER: (your database username from Step 2)
  □ Update DB_PASS: (your database password from Step 2)
  □ Save file

  □ Set folders to 755:

    □ modules
