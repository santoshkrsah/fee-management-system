🚀 HOW TO USE (3 Options)
═══════════════════════════════════════════════════════════════════════

OPTION 1: PWA (Easiest - No APK needed)
───────────────────────────────────────────────────────────────────────

1. Upload these files to your Hostinger:
   □ manifest.json → public_html/
   □ sw.js → public_html/
   □ includes/pwa_head.php → public_html/includes/
   □ App icons → public_html/assets/images/

2. Add PWA code to your PHP pages (one line):
   <?php include '../includes/pwa_head.php'; ?>

3. Enable HTTPS on Hostinger
   (Security → SSL → Enable)

4. Users visit your website on mobile Chrome
   → Tap "Install app" or "Add to Home Screen"
   → App appears on home screen!

Time Required: 15 minutes
Best For: Quick deployment, no technical knowledge

OPTION 2: APK via PWA Builder (Recommended)
───────────────────────────────────────────────────────────────────────

1. Complete Option 1 first (PWA setup)

2. Go to: https://www.pwabuilder.com/

3. Enter your website URL

4. Click "Package For Stores" → Android

5. Fill in app details and generate

6. Download APK (FREE!)

7. Distribute to users

Time Required: 5 minutes (after PWA setup)
Best For: Professional APK, no coding required

OPTION 3: Automated APK Generation
───────────────────────────────────────────────────────────────────────

1. Complete Option 1 first (PWA setup)

2. Install Node.js on your computer

3. Run the script:
   cd /Users/santosh.sah/repos/WA
   ./generate_apk.sh

4. Follow prompts

5. APK generated automatically

Time Required: 10 minutes
Best For: Tech users, custom builds

⚠️ IMPORTANT NOTES
═══════════════════════════════════════════════════════════════════════

HTTPS IS REQUIRED:
- PWAs only work over HTTPS
- No HTTPS = No PWA features
- Enable SSL certificate on Hostinger first

Icon Files Must Exist:
- All 8 icon sizes must be uploaded
- Must be PNG format
- Must be at exact paths in manifest.json

Service Worker Location:
- sw.js must be in ROOT directory (public_html/)
- Not in subdirectory
- Must be accessible at: yourdomain.com/sw.js

Manifest Location:
- manifest.json must be in ROOT directory
- Must be accessible at: yourdomain.com/manifest.json
