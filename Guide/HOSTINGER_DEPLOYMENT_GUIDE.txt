# Hostinger Deployment Guide - Fee Management System
## Simple Step-by-Step Instructions (For Beginners)

---

## PART 1: Get Your Files Ready

### What You'll Need:
1. Hostinger hosting account
2. All your website files (in the fee_management_system folder)
3. FileZilla (a free FTP program) or use Hostinger File Manager

---

## PART 2: Login to Hostinger

1. Go to https://hostinger.com
2. Click "Login" at the top
3. Enter your email and password
4. You'll see your "Control Panel" (called hPanel)

---

## PART 3: Create Your Database

### Step 3.1: Open Database Manager
1. In hPanel, look for "Databases" section
2. Click on "MySQL Databases"
3. Click "Create New Database"

### Step 3.2: Fill Database Details
1. **Database Name:** Type `fee_management` (or any name you like)
2. **Database Username:** Type `feeuser` (or any name you like)
3. **Password:** Create a strong password (WRITE THIS DOWN!)
4. Click "Create"

### Step 3.3: Note Down Your Details
**IMPORTANT - SAVE THESE DETAILS:**
```
Database Host: localhost
Database Name: u123456789_fee_management  (Hostinger adds prefix automatically)
Database Username: u123456789_feeuser     (Hostinger adds prefix automatically)
Database Password: [The password you just created]
```

---

## PART 4: Upload Your Database

### Step 4.1: Open phpMyAdmin
1. In hPanel, go to "Databases" section
2. Click on "phpMyAdmin"
3. Login (username and password from Step 3.3)

### Step 4.2: Import Database
1. On the left side, click on your database name (u123456789_fee_management)
2. Click the "Import" tab at the top
3. Click "Choose File" button
4. Select the file: **database_complete_with_students.sql** (from your computer)
5. Scroll down and click "Import" button
6. Wait for "Import has been successfully finished" message
7. You should see 11 tables created with 50 students data

---

## PART 5: Upload Your Website Files

### Option A: Using Hostinger File Manager (Easiest)

1. In hPanel, go to "Files" section
2. Click "File Manager"
3. Navigate to "public_html" folder (this is your website root)
4. **DELETE** the default files (index.html, etc.) - Select all → Delete
5. Click "Upload" button at the top
6. Select ALL files from your fee_management_system folder
7. Wait for upload to complete

### Option B: Using FileZilla (Alternative)

1. Download FileZilla from https://filezilla-project.org
2. Get FTP details from Hostinger:
   - In hPanel → Files → FTP Accounts
   - Copy: Host, Username, Password, Port
3. Open FileZilla
4. Enter FTP details at the top
5. Click "Quickconnect"
6. On RIGHT side: Navigate to public_html folder
7. On LEFT side: Select your fee_management_system folder
8. Drag and drop all files from left to right
9. Wait for upload (may take 5-10 minutes)

---

## PART 6: Update Database Configuration

### Step 6.1: Edit Configuration File
1. In File Manager, open: **public_html/config/database.php**
2. Click "Edit" button (or right-click → Edit)
3. Find these lines (around line 10-14):

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'fee_management_system');
define('DB_USER', 'root');
define('DB_PASS', 'Te@5219981998');
```

4. **CHANGE** them to your Hostinger details from Step 3.3:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'u123456789_fee_management');     // Your database name
define('DB_USER', 'u123456789_feeuser');             // Your username
define('DB_PASS', 'YourPasswordFromStep3');          // Your password
```

5. Click "Save" and close

---

## PART 7: Set Correct File Permissions

1. In File Manager, select these folders:
   - assets
   - uploads (if exists)
   - includes
2. Right-click → "Permissions" or "Change Permissions"
3. Set to: **755** (or check: Owner[Read,Write,Execute], Group[Read,Execute], Public[Read,Execute])
4. Click "Apply"

---

1. Go to your domain: **http://yourdomain.com**

## PART 9: Important Security Steps

### After Everything Works:

1. **Change Admin Password Immediately**
   - Login → Admin Settings → Change Password

2. **Delete Test File**
   - Delete `generate_students.php` from public_html

3. **Enable SSL (HTTPS)**
   - In hPanel → Security → SSL
   - Enable "Force HTTPS"
   - Your site will be: https://yourdomain.com

4. **Regular Backups**
   - In hPanel → Files → Backups
   - Create weekly backups

## Troubleshooting Common Issues

### Issue 1: "Database Connection Failed"
**Solution:**
- Check database.php file has correct details
- Verify database name, username, password
- Make sure database was created successfully

### Issue 2: Blank White Page
**Solution:**
- Check file permissions (should be 755 for folders, 644 for files)
- Enable error display: Add to top of index.php:
  ```php
  error_reporting(E_ALL);
  ini_set('display_errors', 1);
  ```

### Issue 3: Can't Upload Files
**Solution:**
- Check your Hostinger plan has enough space
- Try uploading in smaller batches
- Use FileZilla instead of File Manager

### Issue 4: CSS/Images Not Loading
**Solution:**
- Check if assets folder was uploaded
- Verify file permissions on assets folder (755)
- Clear browser cache (Ctrl+Shift+Delete)
