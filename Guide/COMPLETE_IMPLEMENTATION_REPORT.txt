## HOW TO USE NEW FEATURES

### 1. Email Configuration
**Location:** /config/email_config.php

**Setup for Gmail:**
```php
define('SMTP_HOST', 'smtp.gmail.com');
define('SMTP_PORT', 587);
define('SMTP_USERNAME', 'your-email@gmail.com');
define('SMTP_PASSWORD', 'your-16-char-app-password');
```

**Get Gmail App Password:**
1. Enable 2-factor auth on Google account
2. Visit: https://myaccount.google.com/apppasswords
3. Generate app password
4. Use that in SMTP_PASSWORD

### 2. Send Email from Code
```php
require_once '../includes/email_helper.php';

// Send fee receipt
EmailHelper::sendFeeReceipt('parent@email.com', $receiptData);

// Send password reset
EmailHelper::sendPasswordReset('user@email.com', [
    'name' => 'User Name',
    'reset_link' => $resetLink
]);

// Send test email
EmailHelper::sendTest('test@email.com');
```

### 3. Generate PDF Receipt
```php
require_once '../includes/pdf_helper.php';

// Generate and download
PDFHelper::generateFeeReceipt($receiptData, true);

// Or just display
PDFHelper::generateFeeReceipt($receiptData, false);
```

### 4. Create Database Backup
**Access:** Login → Settings → Database Backup
**Action:** Click "Download Backup Now"
**Result:** Downloads complete database as .sql file

### 5. Restore Database
**Access:** Login → Settings → Database Backup
**Action:** Upload → Select .sql file → Restore
**Warning:** THIS WILL OVERWRITE ALL DATA!

### 6. View Audit Logs
**Access:** Login → Settings → Audit Log
**Features:**
- Filter by user, action, date
- View detailed changes
- Export logs (if needed)

### 7. Change Password
**Access:** Click your name → Change Password
**Requirements:**
- Current password
- New password (8+ chars, mixed)
- Confirmation

### 8. Forgot Password
**Access:** Login page → "Forgot Password?"
**Process:**
1. Enter email
2. Receive reset link (email or on-screen)
3. Click link
4. Enter new password
5. Done!
